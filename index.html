¡Bienvenidos!

Tutorial de como usar la Pokedex:

  • Busca el nombre del pokémon que quieras y luego pulsa "Agregar al Equipo"
  • Podrás ver el nombre, el ID y el tipo
  • Puedes utilizar las flechas de la pokedex para cambiar las imagenes de los pokémon
  • Puedes resetear el equipo armado pulsando el boton "Resetear Equipo"
